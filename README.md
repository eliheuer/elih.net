# elih.net

Source for [elih.net](https://elih.net), Eli Heuer's personal site and blog. Built with [Astro](https://astro.build) and deployed to GitHub Pages.

## Develop

```sh
pnpm install
pnpm dev      # local dev server at http://localhost:4321
pnpm build    # production build (runs astro check first)
```

Pushing to `main` triggers the GitHub Pages deploy in `.github/workflows/deploy.yml`.
